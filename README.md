# homm3

First draft Phaser 3 battlefield skeleton inspired by Heroes of Might and Magic III.

## Run locally

Open `index.html` in a browser, or serve the repository with any static file server.

## What is included

- Pointy-top hexagonal battlefield grid
- Placeholder units on the grid
- Click a unit to select it
- Click an empty hex to move the selected unit

## Deployment

GitHub Actions workflow at `.github/workflows/deploy-pages.yml` deploys the game to GitHub Pages on pushes to `main`.
